# 序列模型应用详细文档

## 1. 文本生成

### 1.1 应用

- 诗歌创作
- 代码生成
- 对话系统

### 1.2 技术

- 字符级/词级RNN
- 温度采样
- Beam Search

## 2. 时间序列预测

### 2.1 应用

- 股票预测
- 天气预测
- 能源消耗

### 2.2 技术

- 多步预测
- 滑动窗口
- 注意力机制

## 3. 机器翻译

### 3.1 应用

- 语言互译
- 文档翻译
- 实时翻译

### 3.2 技术

- Encoder-Decoder
- 注意力机制
- Transformer

## 4. 总结

序列模型应用广泛，Transformer成为主流。
