# LSTM详细文档

## 1. 概念介绍

### 1.1 什么是LSTM

LSTM (Long Short-Term Memory) 是RNN的一种变体，通过门控机制解决长序列梯度问题。

### 1.2 核心思想

通过三个门（遗忘门、输入门、输出门）控制信息的流动。

## 2. 门控机制

### 2.1 遗忘门

决定丢弃什么信息:
```
f_t = σ(W_f · [h_{t-1}, x_t])
```

### 2.2 输入门

决定存储什么新信息:
```
i_t = σ(W_i · [h_{t-1}, x_t])
C̃_t = tanh(W_C · [h_{t-1}, x_t])
```

### 2.3 输出门

决定输出什么信息:
```
o_t = σ(W_o · [h_{t-1}, x_t])
h_t = o_t * tanh(C_t)
```

## 3. 优点

- 解决长序列梯度消失
- 长期依赖建模
- 灵活的门控机制

## 4. 总结

LSTM是序列建模的重要突破。
