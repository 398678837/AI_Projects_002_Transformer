# GRU详细文档

## 1. 概念介绍

### 1.1 什么是GRU

GRU (Gated Recurrent Unit) 是LSTM的简化版本，参数更少，但性能相当。

### 1.2 与LSTM对比

- GRU: 3个门 (更新门、重置门、输出门)
- LSTM: 4个门 (遗忘门、输入门、候选记忆、输出门)

## 2. 门控机制

### 2.1 更新门

控制过去信息保留多少:
```
z_t = σ(W_z · [h_{t-1}, x_t])
```

### 2.2 重置门

控制忽略多少过去信息:
```
r_t = σ(W_r · [h_{t-1}, x_t])
```

### 2.3 候选隐藏状态

```
h̃_t = tanh(W_h · [r_t * h_{t-1}, x_t])
```

### 2.4 最终隐藏状态

```
h_t = (1 - z_t) * h_{t-1} + z_t * h̃_t
```

## 3. 优点

- 参数少，训练快
- 性能与LSTM相当
- 结构更简单

## 4. 总结

GRU是LSTM的有效替代方案。
