# RNN详细文档

## 1. 概念介绍

### 1.1 什么是RNN

循环神经网络(Recurrent Neural Network, RNN)是一种用于处理序列数据的神经网络。

### 1.2 核心思想

通过隐藏状态在时间步之间传递信息。

## 2. 技术原理

### 2.1 前向传播

```
h_t = tanh(W_xh * x_t + W_hh * h_{t-1} + b_h)
y_t = W_hy * h_t + b_y
```

### 2.2 时间展开

RNN沿时间步展开，权重共享。

## 3. 问题

### 3.1 梯度消失

- 长序列难以学习
- 早期信息丢失

### 3.2 梯度爆炸

- 训练不稳定

## 4. 总结

RNN是序列建模的基础，但存在梯度问题。
