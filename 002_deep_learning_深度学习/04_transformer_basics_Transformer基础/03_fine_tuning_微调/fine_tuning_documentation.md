# 微调详细文档

## 1. 概念介绍

### 1.1 什么是微调

微调是在预训练模型基础上，在下游数据上继续训练。

### 1.2 为什么微调

- 预训练模型学习通用知识
- 微调适配特定任务
- 减少数据需求

## 2. 微调策略

### 2.1 全参数微调

- 更新所有参数
- 性能最好
- 需要更多GPU

### 2.2 冻结微调

- 冻结底层参数
- 只训练顶层
- 资源需求少

### 2.3 LoRA

- 低秩矩阵适配
- 只更新少量参数
- 效率高

### 2.4 Prefix-Tuning

- 添加可学习前缀
- 不改变原模型
- 参数量极小

## 3. 总结

微调是应用预训练模型的关键步骤。
