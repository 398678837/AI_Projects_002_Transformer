# 迁移学习详细文档

## 1. 概念介绍

### 1.1 什么是迁移学习

迁移学习(Transfer Learning)是将预训练模型的知识迁移到新任务的技术。

### 1.2 优势

- 减少训练数据需求
- 加快训练速度
- 提高模型性能

## 2. 常见策略

### 2.1 特征提取

- 冻结预训练模型所有层
- 训练新的分类器
- 适合数据较少的场景

### 2.2 微调 (Fine-tuning)

- 解冻部分顶层
- 联合训练
- 适合数据中等的场景

### 2.3 完全训练

- 解冻所有层
- 从头训练
- 适合数据充足的场景

## 3. 常用预训练模型

| 模型 | 来源 | 特点 |
|------|------|------|
| ResNet | ImageNet | 通用 |
| VGG | ImageNet | 简单 |
| MobileNet | ImageNet | 轻量 |
| BERT | 文本 | NLP |

## 4. 总结

迁移学习是深度学习中的重要技术。
