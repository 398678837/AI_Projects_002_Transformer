# 池化层详细文档

## 1. 概念介绍

### 1.1 什么是池化层

池化层(Pooling Layer)用于减小特征图的空间尺寸，同时保留重要信息。

### 1.2 作用

- 减少参数数量
- 减少计算量
- 防止过拟合
- 提供平移不变性

## 2. 常见池化方法

### 2.1 最大池化 (Max Pooling)

- 取区域内的最大值
- 保留最显著的特征

### 2.2 平均池化 (Average Pooling)

- 取区域内的平均值
- 平滑特征

### 2.3 全局池化 (Global Pooling)

- 对整个特征图池化
- 减少全连接层参数

## 3. 参数

| 参数 | 说明 |
|------|------|
| 池化大小 | 通常2x2 |
| 步长 | 通常等于池化大小 |

## 4. 总结

池化层是CNN中的重要组件，用于降维和特征提取。
