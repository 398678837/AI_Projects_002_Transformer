# 卷积层详细文档

## 1. 概念介绍

### 1.1 什么是卷积层

卷积层是CNN的核心，通过卷积核在输入图像上滑动提取特征。

### 1.2 卷积操作

```
output[i,j] = Σ Σ input[i+m, j+n] * kernel[m,n]
```

## 2. 核心概念

### 2.1 卷积核 (Kernel)

- 小型权重矩阵 (如3x3, 5x5)
- 每个卷积核提取一种特征

### 2.2 步长 (Stride)

- 卷积核移动的步长
- 影响输出尺寸

### 2.3 填充 (Padding)

- 保持空间维度
- 防止边缘信息丢失

### 2.4 特征图 (Feature Map)

- 卷积输出
- 表示提取的特征

## 3. 常见卷积核

| 类型 | 作用 |
|------|------|
| 边缘检测 | 提取边缘 |
| 模糊 | 平滑图像 |
| 锐化 | 增强细节 |

## 4. 总结

卷积层是CNN提取特征的核心组件。
