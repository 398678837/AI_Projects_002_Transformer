# 经典CNN架构详细文档

## 1. LeNet-5 (1998)

### 1.1 结构

- 输入: 32x32 灰度图
- C1: 6个5x5卷积核
- S2: 2x2池化
- C3: 16个5x5卷积核
- S4: 2x2池化
- C5: 120个5x5卷积
- F6: 84个全连接
- 输出: 10

### 1.2 贡献

- 首次在手写数字识别上成功
- 奠定了CNN基础

## 2. AlexNet (2012)

### 2.1 结构

- 输入: 224x224 RGB图
- 5个卷积层
- 3个全连接层
- 参数: 60M

### 2.2 创新

- ReLU激活函数
- Dropout
- GPU并行训练
- 数据增强

## 3. VGG (2014)

### 3.1 结构

- VGG16: 16层
- VGG19: 19层

### 3.2 特点

- 统一使用3x3卷积
- 深度更深
- 参数较多

## 4. ResNet (2015)

### 4.1 创新

- 残差连接: y = F(x) + x
- 解决梯度消失问题

### 4.2 版本

- ResNet-50
- ResNet-101
- ResNet-152

## 5. 总结

从LeNet到ResNet，CNN架构不断进化。
