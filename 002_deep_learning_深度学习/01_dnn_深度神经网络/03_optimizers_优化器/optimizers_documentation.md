# 优化器详细文档

## 1. 概念介绍

### 1.1 什么是优化器

优化器是用于更新神经网络参数的算法，通过最小化损失函数来提高模型性能。

### 1.2 作用

- 计算梯度
- 更新参数
- 加速收敛

## 2. 常见优化器

### 2.1 SGD (随机梯度下降)

- 每次使用一个样本计算梯度
- 简单但收敛慢

### 2.2 Momentum

- 引入动量概念
- 加速收敛，减少震荡

### 2.3 AdaGrad

- 自适应学习率
- 适合稀疏特征

### 2.4 RMSprop

- 使用指数衰减
- 适合非平稳目标

### 2.5 Adam

- 结合Momentum和RMSprop
- 当前最流行的优化器

## 3. 选择指南

| 场景 | 推荐 |
|------|------|
| 默认 | Adam |
| 稀疏数据 | AdaGrad |
| RNN | RMSprop |
| 追求速度 | SGD+Momentum |

## 4. 总结

选择合适的优化器对训练效率和模型性能很重要。
