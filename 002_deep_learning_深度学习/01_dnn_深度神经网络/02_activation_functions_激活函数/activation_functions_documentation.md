# 激活函数详细文档

## 1. 概念介绍

### 1.1 什么是激活函数

激活函数是神经网络中的非线性函数，将线性变换转换为非线性输出。

### 1.2 作用

- 引入非线性
- 控制输出范围
- 决定神经元是否激活

## 2. 常见激活函数

### 2.1 Sigmoid

- 输出范围: (0, 1)
- 公式: f(x) = 1/(1+e^-x)
- 优点: 概率解释
- 缺点: 梯度消失

### 2.2 Tanh

- 输出范围: (-1, 1)
- 公式: f(x) = tanh(x)
- 优点: 零中心化
- 缺点: 梯度消失

### 2.3 ReLU

- 输出范围: [0, ∞)
- 公式: f(x) = max(0, x)
- 优点: 高效、缓解梯度消失
- 缺点: Dying ReLU

### 2.4 Leaky ReLU

- 解决Dying ReLU问题
- 公式: f(x) = max(0.01x, x)

### 2.5 Softmax

- 多分类输出
- 输出概率分布

## 3. 选择指南

| 场景 | 推荐 |
|------|------|
| 二分类 | Sigmoid |
| 多分类 | Softmax |
| 隐藏层 | ReLU |
| 深层网络 | Leaky ReLU / ELU |

## 4. 总结

选择合适的激活函数对网络训练很重要。
