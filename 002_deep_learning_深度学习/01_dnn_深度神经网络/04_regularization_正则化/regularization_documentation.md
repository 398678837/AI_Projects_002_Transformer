# 正则化详细文档

## 1. 概念介绍

### 1.1 什么是正则化

正则化是防止神经网络过拟合的技术，通过在损失函数中添加惩罚项来约束模型复杂度。

### 1.2 过拟合问题

- 训练误差小，测试误差大
- 模型过于复杂
- 泛化能力差

## 2. 常见正则化方法

### 2.1 L1正则化 (Lasso)

- 损失函数: L = 原始损失 + λ||w||₁
- 特点: 产生稀疏权重

### 2.2 L2正则化 (Ridge)

- 损失函数: L = 原始损失 + λ||w||₂²
- 特点: 权重衰减

### 2.3 Dropout

- 训练时随机丢弃神经元
- 相当于集成多个网络

### 2.4 Early Stopping

- 监控验证集损失
- 过拟合前停止训练

### 2.5 数据增强

- 增加训练数据多样性
- 最有效的正则化

## 3. 选择指南

| 方法 | 优点 | 缺点 |
|------|------|------|
| L1 | 稀疏化 | 不常用 |
| L2 | 稳定 | 计算简单 |
| Dropout | 效果好 | 需要调参 |

## 4. 总结

正则化是深度学习中的重要技术。
