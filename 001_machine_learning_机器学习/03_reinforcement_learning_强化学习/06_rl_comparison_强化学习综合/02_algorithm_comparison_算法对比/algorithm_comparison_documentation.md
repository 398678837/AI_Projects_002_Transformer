# 强化学习算法对比详细文档

## 1. 算法分类

### 1.1 按策略类型分类

| 类型 | 代表算法 | 特点 |
|------|----------|------|
| Off-Policy | Q-Learning, DQN | 可从旧数据学习 |
| On-Policy | SARSA, PPO | 只从当前策略学习 |

### 1.2 按方法分类

| 分类 | 算法 | 特点 |
|------|------|------|
| 动态规划 | 策略迭代, 值迭代 | 需要模型 |
| 蒙特卡洛 | MC控制 | 无模型 |
| 时序差分 | Q-Learning, SARSA | 无模型 |
| 深度强化学习 | DQN, PPO | 深度网络 |

## 2. 算法对比

### 2.1 样本效率

| 算法 | 样本效率 | 说明 |
|------|----------|------|
| DQN | 高 | 经验回放 |
| PPO | 中等 | On-Policy |
| Q-Learning | 高 | Off-Policy |
| SARSA | 低 | On-Policy |

### 2.2 稳定性

| 算法 | 稳定性 | 说明 |
|------|----------|------|
| PPO | 高 | 裁剪机制 |
| DQN | 中等 | 目标网络 |
| Q-Learning | 中等 | 收敛保证 |

## 3. 选择指南

### 3.1 如何选择算法

1. **离散动作**: DQN, Double DQN
2. **连续动作**: PPO, SAC
3. **安全关键**: SARSA, PPO
4. **样本效率**: DQN, SAC
5. **稳定性**: PPO

## 4. 总结

根据具体问题选择合适的算法。
