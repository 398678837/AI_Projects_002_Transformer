# 策略梯度教材

## 第一章：策略梯度简介

### 1.1 什么是策略梯度

策略梯度是直接优化策略参数的算法。

### 1.2 核心思想

直接计算梯度并更新策略参数。

## 第二章：策略梯度定理

### 2.1 公式

$$ \nabla_\theta J(\theta) = \mathbb{E}[\nabla_\theta \log \pi_\theta(a|s) \cdot G_t] $$

## 第三章：REINFORCE算法

### 3.1 流程

```
1. 采样轨迹
2. 计算回报
3. 更新策略参数
```

## 第四章：优点与缺点

### 4.1 优点

- 可学习随机策略
- 适合连续动作

### 4.2 缺点

- 方差高
- 局部最优

## 第五章：总结

策略梯度是强化学习的重要组成部分。
