# LDA（线性判别分析）降维详细文档

## 1. 概念介绍

### 1.1 什么是LDA
线性判别分析（Linear Discriminant Analysis，LDA）是一种**有监督的降维算法**，同时也可以用于分类。它的目标是找到一个投影方向，使得不同类别的数据在投影后尽可能分开，同一类别的数据尽可能紧凑。

### 1.2 核心思想
- **类别可分性最大化**：最大化不同类别之间的距离
- **类别紧凑性最大化**：最小化同一类别内部的距离
- **Fisher准则**：最大化Fisher准则函数

### 1.3 应用场景
- **有监督降维**：在有标签的情况下进行降维
- **分类任务**：直接用于分类
- **特征提取**：提取最具判别性的特征
- **与PCA对比**：在有标签的情况下通常比PCA更好

## 2. 技术原理

### 2.1 基本思想

LDA的目标是找到一个投影方向，使得投影后：
- 不同类别的中心尽可能远离
- 同一类别的样本尽可能紧凑

### 2.2 散度矩阵

- **类内散度矩阵**：衡量同一类别内部的离散程度
- **类间散度矩阵**：衡量不同类别之间的离散程度

### 2.3 重要特点

- **有监督**：需要类别标签
- **最大维度**：最多降到n_classes-1维
- **线性变换**：是线性降维方法
- **可用于分类**：本身也是一个分类器

## 3. 代码实现

```python
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis

lda = LinearDiscriminantAnalysis(n_components=2)
X_lda = lda.fit_transform(X, y)
```

## 4. LDA vs PCA

| 特性 | LDA | PCA |
|------|-----|-----|
| 监督性 | 有监督 | 无监督 |
| 目标 | 最大化类别可分性 | 最大化方差 |
| 最大维度 | n_classes-1 | 原维度 |
| 需要标签 | 需要 | 不需要 |
| 分类能力 | 可以分类 | 不能分类 |
