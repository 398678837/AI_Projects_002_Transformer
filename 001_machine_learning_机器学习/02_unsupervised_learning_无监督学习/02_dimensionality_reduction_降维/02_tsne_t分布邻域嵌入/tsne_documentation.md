# t-SNE（t分布邻域嵌入）降维算法详细文档

## 1. 概念介绍

### 1.1 什么是t-SNE
t-SNE（t-distributed Stochastic Neighbor Embedding）是一种**非线性降维算法**，由 Laurens van der Maaten 和 Geoffrey Hinton 于2008年提出。它主要用于数据可视化，能够将高维数据映射到低维空间（通常是2维或3维），同时保留数据的局部结构。

### 1.2 核心思想
- **保持局部相似性**：在高维空间中距离相近的点，在低维空间中也应该距离相近
- **概率分布建模**：在高维空间和低维空间都使用概率分布来描述点之间的相似性
- **t-分布**：在低维空间使用t-分布来缓解拥挤问题
- **梯度下降优化**：通过最小化两个分布之间的K-L散度来优化

### 1.3 应用场景
- **数据可视化**：将高维数据降维到2维或3维，便于直观观察数据结构
- **探索性数据分析**：发现数据中的聚类、离群点等模式
- **生物信息学**：可视化基因表达数据、蛋白质结构等
- **图像处理**：可视化图像特征
- **自然语言处理**：可视化词向量、文档向量

## 2. 技术原理

### 2.1 高维空间的概率分布

在高维空间中，t-SNE使用高斯分布来描述点之间的相似性：

$$ p_{j|i} = \frac{\exp(-\|x_i - x_j\|^2 / 2\sigma_i^2)}{\sum_{k \neq i} \exp(-\|x_i - x_k\|^2 / 2\sigma_i^2)} $$

其中：
- $\sigma_i$ 是点 $i$ 的高斯分布的方差
- 困惑度（perplexity）与 $\sigma_i$ 相关

### 2.2 对称的联合概率分布

为了简化计算，使用对称的联合概率分布：

$$ p_{ij} = \frac{p_{j|i} + p_{i|j}}{2n} $$

### 2.3 低维空间的概率分布

在低维空间中，t-SNE使用t-分布（自由度为1）来描述点之间的相似性：

$$ q_{ij} = \frac{(1 + \|y_i - y_j\|^2)^{-1}}{\sum_{k \neq l} (1 + \|y_k - y_l\|^2)^{-1}} $$

### 2.4 目标函数

t-SNE的目标是最小化高维分布和低维分布之间的K-L散度：

$$ C = KL(P \| Q) = \sum_{i \neq j} p_{ij} \log \frac{p_{ij}}{q_{ij}} $$

### 2.5 梯度下降

使用梯度下降来优化目标函数：

$$ \frac{\partial C}{\partial y_i} = 4 \sum_{j} (p_{ij} - q_{ij}) (y_i - y_j) (1 + \|y_i - y_j\|^2)^{-1} $$

## 3. 重要参数

### 3.1 perplexity（困惑度）

困惑度是t-SNE最重要的参数之一：
- **含义**：可以理解为"每个点的有效邻居数"
- **推荐范围**：5-50
- **影响**：
  - 较小值：关注局部结构，可能忽略全局结构
  - 较大值：关注全局结构，可能忽略局部细节
  - 通常应该小于样本数

### 3.2 learning_rate（学习率）

学习率控制梯度下降的步长：
- **推荐范围**：100-1000
- **常见问题**：
  - 太大：结果看起来像一个"球"
  - 太小：结果看起来太分散
  - 建议：先尝试200，然后调整

### 3.3 n_iter（迭代次数）

迭代次数控制优化过程的长度：
- **最小值**：250次
- **推荐值**：1000次
- **判断标准**：K-L散度应该在迭代过程中稳定

### 3.4 early_exaggeration（早期夸张）

早期夸张参数在优化的早期阶段放大高维空间中的相似性：
- **默认值**：12.0
- **作用**：帮助自然聚类在早期阶段形成
- **推荐**：保持默认值

```python
from sklearn.manifold import TSNE
from sklearn.preprocessing import StandardScaler

# 标准化数据
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 创建t-SNE模型
tsne = TSNE(
    n_components=2,      # 降维到2维

    verbose=1             # 输出详细信息
)

# 降维
X_tsne = tsne.fit_transform(X_scaled)
```

## 5. t-SNE的优缺点

### 5.1 优点

- **保持局部结构**：能够很好地保持数据的局部邻域关系
- **可视化效果好**：在数据可视化方面表现优秀
- **发现聚类**：能够发现数据中的自然聚类
- **非线性降维**：能够处理非线性数据结构
- **广泛应用**：在多个领域都有成功应用

### 5.2 缺点

- **计算复杂度高**：时间复杂度为O(n²)，不适合大规模数据
- **没有显式的映射函数**：不能将新数据直接映射到低维空间
- **参数敏感**：结果对perplexity和learning_rate等参数敏感
- **随机结果**：每次运行结果可能不同（需要设置random_state）
- **不适合保留全局结构**：主要关注局部结构，可能扭曲全局结构

## 6. 实际应用建议

### 6.1 数据预处理

- **标准化**：t-SNE对特征尺度敏感，应该先标准化
- **降维预处理**：对于非常高维的数据（如>50维），可以先用PCA降维到50维左右，再用t-SNE
- **样本量**：t-SNE不适合太大的样本量（建议<10,000）

### 6.2 参数调优

1. **尝试不同的perplexity**：5、15、30、50
2. **调整learning_rate**：100、200、500
3. **增加n_iter**：如果结果不稳定，增加迭代次数
4. **多次运行**：由于随机性，建议多次运行取最好的结果

### 6.3 结果解读

- **聚类**：观察是否有明显的聚类
- **分离度**：不同类别的分离程度
- **连续性**：数据点的连续性
- **异常值**：识别离群点
