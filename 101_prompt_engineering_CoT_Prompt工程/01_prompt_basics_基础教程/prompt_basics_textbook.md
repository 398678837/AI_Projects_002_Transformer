# Prompt基础教材

## 第一章：什么是Prompt工程

### 1.1 核心原则

- **清晰明确**：指令要清晰，避免歧义
- **结构化**：使用统一格式
- **具体详细**：提供详细要求

### 1.2 基本结构

```
角色：{你要AI扮演的角色}
任务：{具体要做什么}
要求：{格式要求/细节/禁忌}
输出：{期望格式}
```

## 第二章：Prompt类型

### 2.1 零样本Prompt
直接给出任务，不提供示例

### 2.2 单样本Prompt
提供1个示例

### 2.3 少样本Prompt（Few-Shot）
提供2-3个示例
