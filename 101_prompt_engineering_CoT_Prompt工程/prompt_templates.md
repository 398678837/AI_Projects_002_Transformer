# Prompt工程+CoT思维链模板库

## 📝 必背万能模板

### ① 标准指令模板
```
角色：{你要AI扮演的角色}
任务：{具体要做什么}
要求：
1. {格式要求}
2. {细节/禁忌}
输出：{期望格式}
```

### ② Few-Shot 模板
```
任务：{任务}
示例1：
输入：{...}
输出：{...}
示例2：
输入：{...}
输出：{...}
请按示例格式输出：
```

### ③ CoT 思维链模板（最核心）
```

1. 先一步步写出思考过程
2. 最后给出最终答案

```

## 🎯 多场景Prompt模板库

### 1. 通用对话
```
角色：AI助手
任务：回答用户问题
要求：
1. 友好、专业
2. 简洁明了
3. 避免敏感内容
输出：自然语言回答
```

### 2. 代码解释/优化
```
角色：资深Python工程师
任务：解释以下代码
要求：
1. 解释代码功能
2. 分析代码优缺点
3. 提供优化建议
输出：详细解释
```

### 3. 文本总结
```
角色：文本分析师
任务：总结以下文本
要求：
1. 提取核心信息
2. 保持简洁
3. 不超过300字
输出：总结文本
```

### 4. 逻辑推理
```
角色：逻辑学家
任务：解决以下逻辑问题
要求：
1. 分析问题
2. 给出解决方案
3. 解释推理过程
输出：详细解答
```

### 5. 数据处理指令
```
角色：数据科学家
任务：处理以下数据
要求：
1. 清洗数据
2. 分析数据
3. 可视化结果
输出：处理结果
```

### 6. Few-Shot示例
```
任务：将数字转换为中文
示例1：
输入：1
输出：一
示例2：
输入：2
输出：二
请按示例格式输出：
```
