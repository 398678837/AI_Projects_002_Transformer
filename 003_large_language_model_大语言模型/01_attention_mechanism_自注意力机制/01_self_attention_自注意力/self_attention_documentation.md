# 自注意力详细文档

## 1. 概念介绍

### 1.1 什么是自注意力

自注意力(Self-Attention)是Transformer的核心机制，允许序列中的每个位置关注所有其他位置。

### 1.2 核心思想

- Query: 我要找什么
- Key: 哪些位置有信息
- Value: 实际的信息内容

## 2. 计算过程

### 2.1 公式

```
Attention(Q,K,V) = softmax(QK^T/√d_k)V
```

### 2.2 步骤

1. 计算Q与K的点积
2. 缩放（除以√d_k）
3. Softmax归一化
4. 加权求和得到V

## 3. 优点

- 并行计算效率高
- 捕获长距离依赖
- 可解释性强

## 4. 总结

自注意力是现代NLP模型的核心组件。
