# 位置编码详细文档

## 1. 概念介绍

### 1.1 什么是位置编码

位置编码(Positional Encoding)为序列中的每个位置添加唯一的表示，使模型能够感知位置信息。

### 1.2 为什么需要位置编码

- 自注意力不包含位置信息
- 需要显式添加位置信息
- 让模型知道词的顺序

## 2. 编码方法

### 2.1 正弦余弦编码

```
PE(pos,2i) = sin(pos/10000^(2i/d_model))
PE(pos,2i+1) = cos(pos/10000^(2i/d_model))
```

## 3. 优点

- 每个位置唯一编码
- 不同位置有规律性
- 可推广到任意长度

## 4. 总结

位置编码是Transformer的必要组件。
