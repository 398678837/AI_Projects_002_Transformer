# 编码器-解码器架构详细文档

## 1. 概念介绍

### 1.1 什么是编码器-解码器架构

编码器-解码器(Encoder-Decoder)是处理序列到序列任务的经典架构。

### 1.2 架构组成

- 编码器：处理输入序列
- 解码器：生成输出序列

## 2. 架构类型

### 2.1 Seq2Seq

- RNN编码器+解码器
- 早期序列到序列模型

### 2.2 Transformer

- 注意力机制
- 并行计算

### 2.3 BERT vs GPT

- BERT：仅编码器（理解）
- GPT：仅解码器（生成）

## 3. 总结

不同架构适用于不同任务。
