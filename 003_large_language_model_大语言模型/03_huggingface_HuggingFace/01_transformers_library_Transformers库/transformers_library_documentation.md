# Transformers库详细文档

## 1. 概念介绍

### 1.1 什么是Transformers库

Transformers是Hugging Face开发的最流行NLP深度学习库，提供丰富的预训练模型。

### 1.2 主要功能

- 预训练模型加载
- 模型微调
- 推理Pipeline

## 2. 常用模型

| 模型 | 类型 | 用途 |
|------|------|------|
| BERT | Encoder | 理解任务 |
| GPT | Decoder | 生成任务 |
| T5 | Enc-Dec | Seq2Seq |
| Llama | Decoder | 开源生成 |
| Qwen | Decoder | 对话 |

## 3. 总结

Transformers库是NLP开发的必备工具。
